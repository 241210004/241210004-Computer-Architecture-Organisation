# Computer Architecture and Organisation — Laboratory Work

This repository contains laboratory exercises and circuit designs developed as part of the course "Computer Architecture and Organisation". It serves as a central collection of Logisim Evolution circuit projects, reference notes, and example configurations that demonstrate the fundamental digital logic building blocks underlying modern computer design.

## Overview

- **Author:** Aayush Anand (241210004)
- **Course:** Computer Architecture and Organisation
- **Primary Tools:** Logisim Evolution (recommended version: v4.0.0), GDB Debugger

The repository emphasizes practical, hands-on circuit design covering both combinational and sequential logic, including adders, multiplexers, encoders, registers, and their supporting wiring structures. Each `.circ` file is a self-contained Logisim Evolution project that can be opened directly within the application without any additional setup.

## Requirements

Logisim Evolution v4.0.0 or later — available at [https://github.com/logisim-evolution/](https://github.com/logisim-evolution/). The `.circ` files embed project metadata that Logisim Evolution reads to correctly restore component placement, gate parameters, and circuit configurations.
